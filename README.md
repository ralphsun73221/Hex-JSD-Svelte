### 241209：九九乘法表的邏輯是什麼？
- 這個可以好好的紀錄一下，為何程式碼最終會是現在這個樣子

    1. 一個是將原本重複 9 次的 `card-body` 縮減到一次，

    3. 增加一個**函式表達式**用來產生 `card-content` 的內容，
    4. 2 跟 3 都搭配 `{#each}` 來產生出重複使用的內容
